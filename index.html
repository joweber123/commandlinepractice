Common commands

⌘ command

  • "ls" will give you a list of what is in that directory
  • "atom ." opens atom on that directory
  • "⌘+K" will clear your terminal
  • "git status" List the files you've changed and those you still need to add or commit:
  • "git add ." stages files before an actual commit
  • "git commit -a -m "our commit message" " this is our short cut to add everything we are working on and commit at the same time

Practice making branches

  • "git checkout -b branchname" this will make a new branch(in this case named branchname) and switch you to it
  • "git push -u origin branchname" will push the branch we made locally to our github
  • "git checkout branchname" will switch to another branch
  • "git branch" gives you a list of your branches
  • "git branch --delete branchname" deletes that branch in your local repo
